# Base Normativa e Contexto

As diretrizes para padronização, estruturação e transmissão eletrônica de registros no âmbito municipal de São Paulo são regidas pelas seguintes portarias:

## Documentos Oficiais
* **Portaria SMS-SP nº 117/2026 (CDEM-RAC):** Estabelece o modelo, as regras e a forma de transmissão eletrônica do Conjunto de Dados Expandido Municipal.
* **Portaria SMS-SP nº 118/2026 (RISAH):** Define os controles mínimos de completude e consistência para internações no ecossistema municipal.

## Síntese Executiva
Ambas as portarias exigem que os sistemas assistenciais adotem um padrão de interoperabilidade compatível com a RNDS, utilizando a especificação HL7 FHIR R4. Os fornecedores devem garantir a qualidade dos dados, submetendo seus sistemas a um processo formal de homologação antes de operar em produção.
