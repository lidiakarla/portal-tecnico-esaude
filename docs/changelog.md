# Histórico de Versões (Changelog)

Todas as alterações relevantes neste portal técnico e nas especificações de API serão registradas nesta página para garantir a rastreabilidade e o planejamento das equipes de desenvolvimento.

---

## [1.0.0-beta] — 2026-03-23
### Novidades
- **Portal Técnico:** Lançamento da nova estrutura de documentação baseada em **MkDocs Material**.
- **Identidade Visual:** Atualização das cores para o padrão institucional (Teal/Verde) e inclusão do logotipo da SMS-SP.
- **Ambiente de Sandbox:** Disponibilização de página com orientações de acesso, limites de uso e formulário de cadastro.

### Especificações Técnicas (FHIR)
- **Modelos de Informação:** Inclusão das definições iniciais para **CDEM-RAC** (Ambulatorial) e **RISAH** (Hospitalar) em conformidade com o padrão **HL7 FHIR R4**.
- **Central de Downloads:** Estruturação da área para disponibilização do Kit Implementador e Coleções de Postman.

### Governança e Suporte
- **Base Normativa:** Integração dos links oficiais para as Portarias SMS nº 117/2026 e nº 118/2026.
- **Processo de Homologação:** Definição dos 6 estágios (*Gates*) para certificação de sistemas parceiros.
- **Canais de Suporte:** Formalização do fluxo de atendimento via Service Desk e e-mail institucional.

---

## [0.1.0-alpha] — 2026-03-02
### Notas Iniciais
- Estruturação preliminar dos campos de dados para os modelos RAC e CMD.
- Definição do cronograma de transição para o novo padrão e-saúdeSP.

---

### Como ler este Changelog?
As alterações são categorizadas da seguinte forma:
* **Adicionado:** Para novos recursos ou páginas.
* **Alterado:** Para mudanças em recursos existentes.
* **Corrigido:** Para correções de bugs ou erros de documentação.
* **Removido:** Para recursos que foram descontinuados.

> ⚠️ **Aviso:** Alterações que quebrem a compatibilidade (*Breaking Changes*) na API serão destacadas com este aviso e notificadas com antecedência mínima de 30 dias.
