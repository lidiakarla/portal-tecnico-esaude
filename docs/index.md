# Portal Técnico de Implementação: CDEM-RAC e RISAH

Bem-vindo ao canal oficial do **e-saúdeSP** para orientação de integração.

## Visão Geral
Este portal concentra a especificação técnica baseada em **HL7 FHIR R4** para envio de dados ao Município de São Paulo:

* **CDEM-RAC:** Conjunto de Dados Expandido Municipal.
* **RISAH:** Registro de Internação e Saúde Associada à Hospitalização.
