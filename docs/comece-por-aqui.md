# Comece por Aqui (Guia de Início Rápido)

Bem-vindo ao Portal de Integração do e-saúdeSP. Este guia foi criado para orientar fornecedores de sistemas de saúde (Prontuários Eletrônicos, Sistemas Hospitalares e de Gestão) no processo de integração com o ecossistema municipal de São Paulo.

## O que é o projeto?
O objetivo é consolidar os dados de saúde dos cidadãos paulistanos em um Repositório Clínico Municipal único, garantindo a continuidade do cuidado entre a rede pública e privada, utilizando padrões internacionais de interoperabilidade (HL7 FHIR).

## Fluxo de Integração (Passo a Passo)

1. **Entenda a Regra:** Leia a [Base Normativa](base-normativa.md) para conhecer as Portarias SMS nº 117 e 118/2026.
2. **Estude o Modelo:** Verifique as tabelas de dados nos documentos do [CDEM-RAC](cdem-rac.md) e do [RISAH](risah.md).
3. **Desenvolva:** Utilize a [Especificação Técnica](especificacao.md) para mapear os campos do seu sistema para o padrão FHIR R4.
4. **Teste no Sandbox:** Solicite acesso ao nosso [Ambiente de Sandbox](sandbox.md) para validar as primeiras transmissões com dados sintéticos.
5. **Homologue:** Após os testes, siga o rito de [Homologação](homologacao.md) para obter a certificação de conformidade.
6. **Produção:** Com a certificação em mãos, seu sistema estará habilitado para o ambiente de Produção.

## Precisa de Ajuda?
Se você é um desenvolvedor e encontrou dificuldades técnicas, consulte a seção de [Suporte](suporte.md) ou abra um ticket diretamente pelo nosso portal de chamados.
