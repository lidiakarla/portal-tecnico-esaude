    Base Normativa e Governança

    As diretrizes para padronização, estruturação e transmissão eletrônica de registros de saúde no município de São Paulo são regidas pelas portarias abaixo. O cumprimento destas normas é requisito obrigatório para a integração ao ecossistema e-saúdeSP.

    Documentos Oficiais e Legislação

    • 📄 Portaria SMS-SP nº 117/2026 (CDEM-RAC): Estabelece o modelo, as regras e a forma de transmissão eletrônica do Conjunto de Dados Expandido Municipal.
      • Clique aqui para acessar o detalhe da Portaria 117
    • 📄 Portaria SMS-SP nº 118/2026 (RISAH): Define os controles mínimos de completude e consistência para internações no ecossistema municipal.
      • Clique aqui para acessar o detalhe da Portaria 118

    Síntese de Conformidade Técnica

    Ambas as portarias exigem que os sistemas assistenciais adotem um padrão de interoperabilidade compatível com a RNDS (Rede Nacional de Dados de Saúde), utilizando a especificação HL7 FHIR R4.

    Os fornecedores devem garantir a qualidade dos dados, submetendo seus sistemas a um processo formal de Homologação antes de operar em ambiente de produção.
